# Minecraft_cmd_CLI

Generate Minecraft Java Edition command on CLI.

---

# Technology Stack

- TypeScript
- Node.js v25.2.0
- Package Manager: pnpm

# How To Use

Usage:
```
mccmd [options] [command]
```

## help

Type
```
mccmd -h
```
or
```
mccmd help
```
for help.
